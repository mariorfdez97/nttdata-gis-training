# JavaScript & TypeScript para Visores Web (NTT DATA)

Este repositorio contiene la ruta de aprendizaje, apuntes y ejercicios prácticos para el dominio de JavaScript Moderno, TypeScript y React, orientado al desarrollo de aplicaciones GIS.

La estructura de estudio sigue las bases de cursos como los de Fernando Herrera y midudev.

---

## Ruta de Aprendizaje

### Fase 1: JavaScript Moderno (ES6+)
- [x] Sintaxis básica, variables (`const`/`let`) y operadores modernos (`?.`, `??`).
- [x] Métodos de arrays (`map`, `filter`, `find`, `some`, `reduce`).
- [ ] Inmutabilidad, desestructuración y operador spread (`...`).
- [ ] Programación asíncrona (Promesas, `async/await`, peticiones HTTP y control de errores).

### Fase 2: TypeScript
- [ ] Configuración inicial del compilador (`tsconfig.json`).
- [ ] Tipado estático (tipos primitivos, tipos personalizados y `interfaces`).
- [ ] Tipado específico para datos GIS (GeoJSON, Coordenadas, Features).
- [ ] Clases, módulos y genéricos.

### Fase 3: React & GIS
- [ ] Ciclo de vida y estado de componentes (`useState`, `useEffect`).
- [ ] Integración de APIs de mapas (Leaflet / OpenLayers / Mapbox).
- [ ] Gestión del estado global y sincronización con el visor.
- [ ] Conexión con APIs backend y servicios espaciales.
